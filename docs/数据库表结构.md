# 数据库表结构

本文档给技术和 AI 阅读，方便代码审查和 Git diff。业务核对字段时优先查看 `docs/数据库表结构.xlsx`。

## 维护规则

- SQL 初始化源头：`backend/db/init/001_schema.sql`
- 已有数据库升级：`backend/db/migrations/*.sql`
- 技术/AI 文档：`docs/数据库表结构.md`
- 业务核对表：`docs/数据库表结构.xlsx`

新增或修改表结构时，以上四处必须同步。不能只改 SQL，也不能只改 Excel。

## 设计约定

- 数据库：PostgreSQL
- 表名前缀：`pms_`
- 主键：默认使用 `id BIGSERIAL PRIMARY KEY`
- 删除标记：业务主表默认使用 `is_deleted SMALLINT`，`0` 正常，`1` 删除
- 时间字段：统一使用 `TIMESTAMPTZ`，默认 `NOW()`
- 接口返回：后端字段保持数据库命名，前端 API 层转换为页面 record

## 表清单

| 表名 | 中文名 | 说明 |
|---|---|---|
| `pms_user` | 用户表 | 后台用户、登录账号、状态和审计信息 |
| `pms_role` | 角色表 | 角色编码、名称、说明和审计信息 |
| `pms_user_role` | 用户角色关联表 | 用户与角色授权关系 |
| `pms_menu` | 菜单表 | 后台菜单、路由、权限点和显示顺序 |
| `pms_role_menu` | 角色菜单关联表 | 角色与菜单权限授权关系 |
| `pms_work_order` | 运维工单表 | 工单主数据、状态、跟进人和处理时间 |
| `pms_archive_type` | 档案类型表 | 基础档案分类 |
| `pms_archive` | 基础档案表 | 基础档案明细 |
| `pms_op_log` | 操作日志表 | 用户操作审计日志 |
| `pms_access_log` | 访问日志表 | 登录访问记录、登录结果、会话时长和客户端信息 |
| `pms_user_preference` | 用户偏好设置表 | 默认进入页面、列表默认条数和外观兼容配置 |
| `pms_message` | 消息表 | 通知和系统消息，记录接收人、已读状态和跳转路径 |

## pms_user 用户表

| 字段 | 类型 | 约束/默认值 | 说明 |
|---|---|---|---|
| `id` | `BIGSERIAL` | 主键 | 数据库主键 |
| `employee_no` | `VARCHAR(50)` | 必填，唯一 | 工号，登录和列表标识 |
| `real_name` | `VARCHAR(50)` | 必填 | 用户姓名 |
| `phone` | `VARCHAR(20)` | 唯一 | 手机号 |
| `avatar_url` | `VARCHAR(300)` | 可空 | 头像地址 |
| `password` | `VARCHAR(255)` | 必填 | bcrypt 加密密码 |
| `status` | `SMALLINT` | 默认 `1` | 1 启用，0 停用 |
| `first_login` | `SMALLINT` | 默认 `1` | 1 首次登录，0 非首次 |
| `creator_id` | `BIGINT` | 可空 | 创建人 |
| `updater_id` | `BIGINT` | 可空 | 更新人 |
| `is_deleted` | `SMALLINT` | 默认 `0` | 删除标记 |
| `created_at` | `TIMESTAMPTZ` | 默认 `NOW()` | 创建时间 |
| `updated_at` | `TIMESTAMPTZ` | 默认 `NOW()` | 更新时间 |

索引：`idx_user_status_deleted(status, is_deleted)`

## pms_message 消息表

用途：后台通知和系统消息，记录接收人、类型、已读状态和跳转路径。

| 字段 | 类型 | 约束/默认值 | 说明 |
|---|---|---|---|
| `id` | `BIGSERIAL` | 主键 | 数据库主键 |
| `recipient_user_id` | `BIGINT` | 必填，引用 `pms_user.id` | 接收用户 |
| `type` | `VARCHAR(30)` | 必填 | `notification` 通知，`system` 系统消息 |
| `title` | `VARCHAR(120)` | 必填 | 消息标题 |
| `description` | `TEXT` | 必填 | 消息正文 |
| `link_path` | `VARCHAR(200)` | 可空 | 点击后的前端跳转路径 |
| `read_at` | `TIMESTAMPTZ` | 可空 | 已读时间，`NULL` 表示未读 |
| `creator_id` | `BIGINT` | 可空 | 创建人或触发人 |
| `is_deleted` | `SMALLINT` | 默认 `0` | 删除标记 |
| `created_at` | `TIMESTAMPTZ` | 默认 `NOW()` | 创建时间 |
| `updated_at` | `TIMESTAMPTZ` | 默认 `NOW()` | 更新时间 |

索引：`idx_message_recipient_read(recipient_user_id, read_at, is_deleted)`、`idx_message_created_at(created_at)`
